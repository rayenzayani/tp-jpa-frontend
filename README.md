# Frontend – Gestion des personnes

## Description
Ce projet représente la partie frontend d’une application web basée sur une architecture Client / Serveur.

Le frontend consomme une API REST développée en Java JEE (JAX-RS) et permet la gestion des personnes.

## Fonctionnalités
- Affichage de la liste des personnes
- Ajout d’une personne
- Modification d’une personne
- Suppression d’une personne

## Technologies utilisées
- HTML
- CSS
- JavaScript (Fetch API)

## Backend
Le backend est développé en Java JEE avec :
- JAX-RS
- JPA (Hibernate)
- MySQL
- Apache Tomcat 9

⚠️ Le backend doit être lancé localement sur Tomcat pour que le frontend fonctionne correctement.

## Lien de l’API
Exemple : http://localhost:8088/tp_jpa/api/persons/all
